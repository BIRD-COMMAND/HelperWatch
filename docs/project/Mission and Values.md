# Mission and Values

## Mission Statement

HelperWatch is a cognitive scaffolding system designed to empower severely impacted special needs children and their caregivers through accessible, low-cost technology. The system provides real-time, location-aware verbal cues and behavioral monitoring for children requiring near-constant executive function support, bridging the gap between total human-dependent care and maximized functional independence.

## Core Values

### Empowerment Through Independence

The system provides children with the prompts, structure, and emotional support needed to navigate daily routines with reduced human intervention. HelperWatch acts as a tireless, non-judgmental, hyper-personalized cognitive prosthetic — not a replacement for human care, but a tool that fills the gaps when caregivers cannot be at a child's side for every micro-step of every task.

### Caregiver Relief

Parents of severely impacted children are forced into the exhausting roles of taskmasters, security guards, and constant monitors. This erodes the joyful, loving dynamics of parenting. By offloading the mechanical, repetitive task of constant prompting, HelperWatch allows parents to move from "taskmaster" to a purely supportive role, shifting the household dynamic from constant correction to positive reinforcement.

### Radical Accessibility

The system must be financially and technically reachable for any family. This means:

- Utilizing repurposed, commodity consumer hardware rather than proprietary devices.
- Designing "plug-and-play" software that requires zero technical expertise to install and operate.
- Embracing open-source development so that no single corporation owns the rights to a disabled child's autonomy.
- Targeting a total system cost under $100 for a family starting from scratch.

### Ethical Safety by Design

Privacy is not a feature — it is a hard-coded requirement. The system is built around on-device speech-to-text (Moonshine STT), strict cloud data minimization, transit encryption, transient cloud memory processing, and transparent, educator-led risk mitigation. Every architectural decision must pass the test: "If this technology were used by someone with bad intentions, what would it enable?" — and the answer must be "as little as possible."

## What HelperWatch Is — and What It Is Not

HelperWatch is a **supplementary tool** designed to catch the gaps when a parent is cooking dinner, sleeping, or managing other children. It is explicitly **not** a replacement for raw human presence, professional therapy, or medical care.

The system is designed to reduce its own involvement over time through dynamic fading — automatically pulling back prompts as the child demonstrates mastery — because the ultimate success metric is a child who no longer needs the device at all.
