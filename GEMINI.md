# Agent Rules
- **Be Concise:** Omit conversational filler. Provide clear, direct explanations.
- **Budget Actions:** Do not write or execute deep research plans for straightforward tasks. Act directly.
- **No Refactoring:** Fix only what is requested. Do not clean or modify unrelated code unless explicitly asked.
- **Markdown Styling:** Markdown will be displayed on GitHub AND converted using Pandoc for import to Google Docs. Use Markdown styling that is acceptable for both viewing environments.
- **No Images in Markdown:** Images are handled separately.
